# Changelog

All notable changes to Git LFS Delta are recorded here.

## [0.1.0-beta.2] - 2026-07-18

- First published internal beta.
- Native Git LFS custom-transfer client for macOS and x86-64 Linux.
- Resumable content-defined chunk uploads/downloads with verified local caching.
- Forgejo-backed authorization, PostgreSQL metadata, and provider-neutral object storage.
- Explicit schema migrations, production container, and one-command development workflows.
